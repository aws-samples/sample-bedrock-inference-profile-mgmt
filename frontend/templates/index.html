📚 Inference Profile Types

🌍 Global Profiles

Cross-region profiles that route requests to any commercial AWS region globally for optimal performance and availability.

Example: global.anthropic.claude-sonnet-4-...

ARN: arn:aws:bedrock:REGION:ACCOUNT:inference-profile/global.MODEL-ID

🗺️ Regional Profiles

Geographic-specific profiles that route within a defined region (US, EU, APAC, etc.) for data residency requirements.

Example: us.anthropic.claude-3-haiku-..., eu.anthropic.claude-...

ARN: arn:aws:bedrock:REGION:ACCOUNT:inference-profile/GEOGRAPHY.MODEL-ID

📦 Application Profiles

Custom profiles you create to configure specific models with tags and settings for your applications.

Example: web-test-my-profile, prod-chatbot-profile

ARN: arn:aws:bedrock:REGION:ACCOUNT:application-inference-profile/PROFILE-ID

🤖 Foundation Models

Base models provided by AWS and partners, invoked directly without inference profiles.

Example: anthropic.claude-sonnet-4-..., amazon.nova-pro-v1:0

ARN: arn:aws:bedrock:REGION::foundation-model/MODEL-ID
