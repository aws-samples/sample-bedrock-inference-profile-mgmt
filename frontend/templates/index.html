Inference Profile Types Learn More →

Global Profiles

Cross-region profiles that route requests to any commercial AWS region globally for optimal performance and availability.

Example: global.anthropic.claude-sonnet-4-20250514-v1:0
arn:aws:bedrock:us-east-1:ACCOUNT:inference-profile/global.anthropic.claude-sonnet-4-20250514-v1:0

Regional Profiles

Geographic-specific profiles that route within a defined region (US, EU, APAC) for data residency requirements.

Example: us.anthropic.claude-3-5-haiku-20241022-v1:0
arn:aws:bedrock:us-east-1:ACCOUNT:inference-profile/us.anthropic.claude-3-5-haiku-20241022-v1:0

Application Profiles

Custom profiles you create to configure specific models with tags and settings for your applications.

Example: web-test-claude-sonnet-4
arn:aws:bedrock:us-east-1:123456789012:application-inference-profile/abc123def456

Foundation Models

Base models provided by AWS and partners, invoked directly without inference profiles.

Example: anthropic.claude-sonnet-4-20250514-v1:0
arn:aws:bedrock:us-east-1::foundation-model/anthropic.claude-sonnet-4-20250514-v1:0
